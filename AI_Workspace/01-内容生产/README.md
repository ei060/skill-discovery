# 01-内容生产

> 内容创作全流程管理

---

## 📁 目录结构

```
01-内容生产/
├── 选题管理/
│   ├── 01-选题库/          # 存放所有选题
│   ├── 02-待发布/          # 已创作但未发布的内容
│   └── 03-已发布/          # 已发布的内容
└── 素材库/
    ├── 文章素材/           # 文章相关的素材
    ├── 视频素材/           # 视频相关的素材
    └── 图片素材/           # 图片相关的素材
```

---

## 🎯 工作流程

### 1. 选题阶段
1. 在 `01-选题库/` 中创建选题文件
2. 记录选题灵感、目标受众、核心观点
3. 评估选题价值和可行性

### 2. 创作阶段
1. 从 `01-选题库/` 选择选题
2. 在 `02-待发布/` 创建内容目录
3. 按目录结构组织内容文件

### 3. 发布阶段
1. 完成内容创作
2. 发布到各平台
3. 移动目录到 `03-已发布/`

### 4. 数据追踪
1. 记录发布数据（阅读量、互动量等）
2. 分析数据，总结经验
3. 提炼到语义记忆

## 💾 智能存储路由

根据 `CLAUDE.md` 中的规则：

- **文章** → `01-内容生产/选题管理/02-待发布/YYYY-MM-DD-标题/`
- **素材** → `01-内容生产/素材库/[类型]/`

AI 会自动识别内容类型并保存到正确位置。

## 📊 内容数据追踪

发布后需要记录的数据：
- 发布平台
- 发布时间
- 阅读量
- 点赞数
- 转发数
- 评论数
- 收藏数
- 其他互动数据

这些数据会在周日数据维护时录入和分析。

## 🔄 自动化流程

### 时间意图捕获
当你提到 "明天发"、"下周发" 等时间词时，AI 会：
1. 自动添加到任务清单
2. 提醒你按时发布

### 发布状态更新
当你提到 "已发布"、"发布了" 时，AI 会：
1. 自动移动目录到 `03-已发布/`
2. 告知你移动成功

## 📝 使用建议

### 选题管理
- 定期整理选题库
- 删除不再有价值的选题
- 更新有潜力的选题

### 素材管理
- 按主题分类素材
- 添加素材标签
- 定期清理无效素材

### 内容创作
- 每个内容一个独立目录
- 保持目录结构统一
- 及时更新发布记录
