# 语义记忆

## 🧠 什么是语义记忆？

语义记忆是 **提炼的可复用知识和规律**，就像大脑的 "教科书知识"。

### 作用
- 知识可以复用，不用每次重新讨论
- 形成方法论和模式库
- 跨场景应用

## 📋 命名规范

**格式**：`[主题]_[类型].md`

**类型标签**：
- `方法论`：通用的做事方法
- `模式`：可复用的设计模式
- `SOP`：标准操作流程
- `架构`：系统架构设计
- `技巧`：实用的技巧和窍门

**示例**：
- `爆款标题_方法论.md`
- `系统架构_模式.md`
- `内容创作_SOP.md`

## 💡 使用场景

### 应该记录到语义记忆
- ✅ 提炼的通用方法
- ✅ 可复用的设计模式
- ✅ 标准操作流程（SOP）
- ✅ 最佳实践
- ✅ 经验总结

### 不应该记录
- ❌ 具体的事件记录（应该放情景记忆）
- ❌ 必须遵守的规则（应该放强制规则）
- ❌ 临时性的笔记

---

## 🔄 从情景记忆提炼

### 提炼流程
```
情景记忆 → 识别模式 → 语义记忆
```

### 提炼时机
- 每次自我迭代时
- 发现重复模式时（≥3次）
- 完成重大项目后
