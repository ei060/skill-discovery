## 👥 人际资源库

参考 `CLAUDE.md` 中的 **全局资源调度** 功能：

当收到任务时，按以下步骤思考：

### 1. 任务分析
- 这个任务需要什么能力？
- 预计需要多少时间？
- 质量要求如何？

### 2. 资源匹配（按优先级）
- ✅ AI 能高质量完成 → 直接执行
- ✅ 需要人类判断但 AI 可辅助 → AI 起草 + 用户审核
- ⚠️ 用户必须亲自做 → 提醒并协助准备
- 💡 有更合适的人 → 查询人际资源库，推荐委托
- 💰 可以外包 → 建议外包方案

### 3. 决策建议格式
```
**任务**：[任务描述]
**推荐方案**：[谁来做]
**理由**：[为什么这样分配最优]
**预计成本**：[时间/金钱/人情]
```

### 原则
**时间价值优先，能委托的尽量委托**
