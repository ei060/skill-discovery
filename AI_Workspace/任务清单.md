## 📋 使用说明

### 添加任务
直接在相应分类下添加任务，格式：
```markdown
- [ ] 任务描述 - 截止日期 - 标签
```

### 完成任务
勾选完成的任务：
```markdown
- [x] 任务描述 - 完成日期 - 标签
```

**重要**：完成的任务会在会话结束时自动归档

### 任务标签
- `#系统`：系统优化相关
- `#内容`：内容创作相关
- `#学习`：学习成长相关
- `#商业`：商业相关
- `#生活`：生活相关

### 自动归档
- 每次会话结束时，已完成的任务（`[x]`）会自动转移到 `任务归档/2026/` 目录
- 保留完整元数据：类型、来源、创建日期、完成日期、成果
