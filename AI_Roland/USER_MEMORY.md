### 三层记忆架构
1. **情景记忆**: 具体事件和对话（按月归档）
2. **语义记忆**: 提炼的知识和模式
3. **强制规则**: 系统必须遵守的行为约束

### 记忆提炼流程
```
情景记忆 → 识别模式 → 语义记忆
语义记忆 → 发现必须遵守的规则 → 强制规则
```

### 提炼时机
- 每次自我迭代时
- 发现重复模式时（≥3次）
- 完成重大项目后

### 记忆搜索策略
- **关键词提取**: 使用jieba分词
- **时间权重**: 近期记忆权重更高
- **相关性评分**: TF-IDF相似度计算
- **分层搜索**: 先搜索语义记忆，再搜索情景记忆
