## 改进建议

### 优先级：高（立即执行）

#### 1. 修复编码问题 ⚠️

**问题**：
- Windows 控制台使用 GBK 编码
- 导致 emoji 和中文显示异常
- 影响用户体验

**解决方案**：
```python
# 在所有脚本开头添加
import sys
import os
if sys.platform == 'win32':
    import io
    sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8', errors='replace')
    sys.stderr = io.TextIOWrapper(sys.stderr.buffer, encoding='utf-8', errors='replace')
    os.environ['PYTHONIOENCODING'] = 'utf-8'
```

**受影响文件**：
- `system/memory_search.py`
- `system/second_brain.py`
- `system/skills_cli.py`
- 所有命令行工具

**预计时间**：30分钟
**预期收益**：用户体验显著提升

### 优先级：中（本周完成）

#### 3. 升级记忆搜索（可选）

**当前实现**：TF-IDF + jieba 分词

**升级方案**：Sentence Transformers

**优势**：
- 🚀 语义理解更强
- 🚀 向量化检索
- 🚀 多语言支持

**实现步骤**：
```python
# 安装
pip install sentence-transformers

# 使用
from sentence_transformers import SentenceTransformer
model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2')

# 搜索
embeddings = model.encode(documents)
query_emb = model.encode(query)
similarities = cosine_similarity(query_emb, embeddings)
```

**预计时间**：2-3小时
**预期收益**：搜索准确度提升 20-30%

**风险**：
- 首次加载慢（~5秒）
- 内存占用增加（~500MB）
- 需要模型文件（~400MB）

**建议**：作为可选功能，保留 TF-IDF 作为降级方案
