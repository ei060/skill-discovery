## 2. 统一记忆系统原理

### 2.1 核心概念：本能 (Instinct)

**本能**是系统的最小学习单位，代表一个"如果...那么..."的行为模式：

```yaml
---
id: prefer-functional-style
trigger: "when writing new functions"
confidence: 0.7
domain: "code-style"
scope: project
project_id: a1b2c3d4e5f6
lifecycle_stage: sprout
---

# Prefer Functional Style

## Action
Use functional patterns over classes when appropriate.

## Evidence
- Observed 5 times in session abc123
- User corrected class-based approach to functional
```

### 2.2 生命周期机制

```
置信度 1.0 ─────────────────────────────────────┐
           │                                     │
           │                                     │
置信度 0.8 ──────── 🌿 绿叶 ─────────────────────┤
           │             ↑                      │
           │             │ +0.08 (使用)          │
置信度 0.7 ──────── 🌱 萌芽 ────┐                │
           │             │       │                │
           │             │       │                │
置信度 0.5 ────      │       │                │
           │           │       │                │
置信度 0.3 ────── 🍁 枯叶 ────┘                │
           │                                     │
置信度 0.0 ───────── 🪨 土壤 ────────────────┘
           │                                     │
           └──── (精华提取，可复苏) ─────────────┘
```

**状态转换规则**:

```python
confidence >= 0.8 → 🌿 绿叶 (已验证的行为)
confidence >= 0.5 → 🍂 黄叶 (使用频率下降)
confidence >= 0.3 → 🌱 萌芽 (新模式)
confidence < 0.3  → 🍁 枯叶 (即将归档)
confidence == 0.0 → 🪨 土壤 (精华保存)
```

### 2.4 置信度机制

**置信度增长**:
- 搜索命中: `+0.03`
- 引用使用: `+0.08`
- 人工标记: `+0.95` (直接设为高置信度)
- 观察确认: `+0.05` (每次模式重现)

**置信度衰减** (每日执行):
- P0 优先级: 不衰减 (核心知识)
- P1 优先级: `-0.004/天`
- P2 优先级: `-0.008/天`

### 4.3 技能搜索算法

```python
def search_skills(query: str) -> List[SkillMetadata]:
    query_lower = query.lower()
    results = []

    for skill in skills.values():
        # 多维度匹配
        if (query_lower in skill.name.lower() or      # 名称匹配
            query_lower in skill.description.lower() or # 描述匹配
            any(query_lower in tag.lower() for tag in skill.tags)):  # 标签匹配
            results.append(skill)

    # 按优先级排序
    return sorted(results, key=lambda x: x.priority)
```

## 8. 关键算法

### 8.1 项目ID生成

```python
def generate_project_id():
    # 1. 尝试获取 git remote URL (跨机器一致性)
    remote_url = git("remote get-url origin")
    if remote_url:
        return sha256(remote_url).hexdigest()[:12]

    # 2. 回退到 git root path (机器特定)
    root_path = git("rev-parse --show-toplevel")
    return sha256(root_path).hexdigest()[:12]
```

### 8.2 置信度计算

```python
def calculate_initial_confidence(observations):
    count = len(observations)

    if count <= 2:
        return 0.3  # 试探性
    elif count <= 5:
        return 0.5  # 中等
    elif count <= 10:
        return 0.7  # 强
    else:
        return 0.85  # 很强
```

def update_lifecycle_stage(confidence):
    if confidence >= 0.8:

## 总结

AI Roland v2.0 的核心原理是：

1. **一切皆观察** - Hook 捕获所有操作
2. **观察即学习** - 自动检测模式
3. **学习需验证** - 置信度机制
4. **知识会进化** - 本能→技能→命令→代理
5. **项目相隔离** - 不同项目的知识分离
6. **全局可共享** - 通用知识全局化

这是一个**自学习的智能系统**，随着使用时间的增长，它会越来越了解用户的偏好和习惯，提供越来越精准的建议和自动化能力。
