## ✅ 命令别名系统 (已完成)

### 实现内容

创建了完整的命令别名系统，提供简短易记的命令别名。

**文件**:
- `system/roland_alias.py` - 别名解析器
- `system/m.bat`, `i.bat`, `s.bat`, `e.bat`, `p.bat` - Windows 快捷方式

### 可用别名

```
记忆管理:
  m, memory, mem        查看记忆系统状态
  o, obs, log           查看观察记录

本能管理:
  i, instincts, list    列出所有本能
  s, search, find       搜索本能

进化与提升:
  e, evolve, up         进化指定本能
  p, promote, upgrade   提升到全局
  b, boost, star        重要标记

系统维护:
  a, analyze            触发观察分析
  d, decay              置信度衰减
  c, cleanup, clean     清理枯萎本能

帮助:
  h, help, ?            显示此帮助
```

### 使用方法

```bash
# 直接使用 Python
python AI_Roland/system/roland_alias.py m
python AI_Roland/system/roland_alias.py s git --json
python AI_Roland/system/roland_alias.py e use-read --boost 0.1

# 使用 Windows 批处理
cd AI_Roland/system
m.bat          # 查看状态
s.bat git      # 搜索
e.bat use-read # 进化
```

### Windows 兼容性

别名系统解决了 Windows 路径自动转换问题（`/m` → `M:/`）：
- 自动去除前导斜杠
- 支持带/不带斜杠的格式
- 正确解析命令参数
