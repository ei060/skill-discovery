## 一、问题分析

### 1.1 错误现象
```
2026-03-27 启动日志：
[ERROR] Browser initialization failed: 'NoneType' object has no attribute 'isatty'
守护进程状态：stopped
最后心跳：超过 24 小时前
```

### 1.2 根本原因

#### 代码层面
```python
# browser_controller.py:861
headless = not sys.stdin.isatty() or not hasattr(sys.stdout, 'isatty')
```

**问题**：
- 在守护进程中，`sys.stdin` 为 `None`
- 直接调用 `None.isatty()` 导致错误
- 缺少 None 检查和异常处理

#### 环境因素
1. **守护进程**: stdin 被重定向到 `/dev/null`
2. **pythonw.exe**: 无控制台，标准流为 None
3. **Windows 服务**: 没有控制台环境
4. **重定向输入**: stdin 不是 TTY

## 二、解决方案

### 2.1 立即修复

#### 修复 browser_controller.py
```python
# 修复前（第 861 行）
headless = not sys.stdin.isatty() or not hasattr(sys.stdout, 'isatty')

# 修复后
try:
    is_atty = sys.stdin is not None and sys.stdin.isatty()
except (AttributeError, OSError):
    is_atty = False
headless = not is_atty or not hasattr(sys.stdout, 'isatty')
```

### 2.2 系统性改进

#### 创建 sys_utils.py 统一安全接口
```python
class SafeIO:
    """安全的系统 I/O 访问"""

    @staticmethod
    def is_interactive() -> bool:
        """检测是否在交互式终端中运行"""
        try:
            if sys.stdin is None or not hasattr(sys.stdin, 'isatty'):
                return False
            if not sys.stdin.isatty():
                return False
            if not hasattr(sys.stdout, 'isatty'):
                return False
            if not sys.stdout.isatty():
                return False
            return True
        except (AttributeError, OSError, ValueError):
            return False

    @staticmethod
    def safe_read_stdin(max_length: int = 10240) -> Optional[str]:
        """安全地从 stdin 读取"""
        if sys.stdin is None:
            return None
        try:
            if hasattr(sys.stdin, 'buffer'):
                raw_bytes = sys.stdin.buffer.read(max_length)
                return raw_bytes.decode('utf-8', errors='replace') if raw_bytes else ""
            else:
                return sys.stdin.read(max_length)
        except (AttributeError, OSError, ValueError, UnicodeDecodeError):
            return None
```

#### 更新所有不安全代码
```python

# task_dispatcher.py - 修复后
```
