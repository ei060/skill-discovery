## 🔧 技术栈

- **语言**: Python 3.10+
- **核心库**:
  - `croniter` - Cron 表达式解析
  - `flask` - HTTP API
  - `flask-cors` - CORS 支持
  - `requests` - HTTP 请求
- **数据格式**: JSON, JSONL, Markdown

## 📝 已知问题

1. **编码显示**: Windows 下部分中文显示为乱码
   - 原因: Windows GBK 编码限制
   - 影响: 仅显示问题，功能正常
   - 解决: 使用 `start.bat` 或设置 `chcp 65001`

2. **engine_v2.py 语法错误**: 包含特殊字符导致编码问题
   - 影响: 无法直接导入
   - 解决: 使用 v1.0 engine.py，或修复编码
