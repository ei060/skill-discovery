### 2. 自动化检测

#### 创建 pre-commit hook
```bash
# .git/hooks/pre-commit
#!/bin/bash
# 检测不安全的 sys.stdin 访问
if git diff --cached --name-only | xargs grep -l "sys.stdin.read()"; then
    echo "警告: 检测到不安全的 sys.stdin 访问"
    echo "请使用 sys_utils.safe_read_stdin()"
    exit 1
fi
```

#### 创建 linter 规则
```python
# pylint checker
def visit_call(self, node):
    if node.func.attrname == 'isatty':
        if not self.is_safe_context(node):
            self.add_message('unsafe-isatty', node=node)
```
