# AI Roland 自动化引擎

> 让三层记忆架构自动运行

## ✨ 自动化功能

### 1. 时间意图自动捕获

**触发词**：
- "明天"、"后天"、"下周"、"X月X日"
- "要发"、"要做"、"记得"、"别忘了"
- "计划"、"打算"、"准备"

**自动执行**：
- 添加到任务清单（根据紧急程度分类）
- 告知用户已添加

**示例**：
```
输入: "这篇文章明天要发布"
结果: 自动添加到【紧急重要】任务
```

### 2. 发布状态自动更新

**触发词**：
- "已发布"、"发布了"、"发出去了"
- "已经发了"、"刚发了"

**自动执行**：
- 移动文章从 `02-待发布/` → `03-已发布/`
- 告知用户已完成

### 3. 智能存储路由

**检测内容类型**：
- 文章 (>500字) → `01-内容生产/选题管理/02-待发布/`
- 研究 → `03-科研/`
- 商业 → `02-商业/`
- 工具 → `04-工具与效率/`
- 素材 → `01-内容生产/素材库/`

**示例**：
```
输入: "帮我写一篇关于LLM的文章"
结果: 自动保存到正确位置
```

### 4. 每日简报

**触发时机**：每天第一次启动时

**内容**：
- 显示所有未完成任务
- 按优先级分组
- 提供任务统计

### 5. 周日数据维护提醒

**触发时机**：每周日

**内容**：
- 提醒录入本周内容数据
- 分析内容表现
- 更新爆款递归库

## 🧠 三层记忆架构

```
记忆库/
├── 情景记忆/    # 具体事件和对话（按月归档）
├── 语义记忆/    # 提炼的知识和规律
└── 强制规则/    # 必须遵守的行为约束
```

### 记忆提炼流程

```
情景记忆 → 识别模式 → 语义记忆
语义记忆 → 发现规则 → 强制规则
```

## 🔧 系统架构

```
system/
├── engine.py        # 核心引擎
├── cli.py           # 命令行界面
├── config.json      # 配置文件（可选）
└── README.md        # 本文档
```

### 核心组件

1. **RolandEngine** - 主控制器
2. **IntentParser** - 意图识别器
3. **FileRouter** - 智能存储路由
4. **MemoryManager** - 记忆管理器
5. **TaskManager** - 任务管理器
6. **Scheduler** - 定时任务调度器

```python
def parse(self, text):
    intents = {}

    # 添加新的模式
    if re.search(r'你的模式', text):
        intents["new_intent"] = text

    return intents
```
