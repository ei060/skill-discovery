## 🔧 工作原理

### 自动启动流程

```
启动 Claude
    ↓
运行启动脚本
    ↓
检查守护进程状态
    ↓
├─ 未运行 → 自动启动
└─ 运行中 → 继续使用
    ↓
加载对话历史（最近3条）
    ↓
加载当前任务
    ↓
生成简报
    ↓
记录新会话
    ↓
开始工作 ✅
```

### 守护进程后台运行

守护进程（daemon.py）会在后台持续运行：

```
每 60 秒：
  ├─ 心跳检测 (#42, #43, ...)
  ├─ 自动更新任务统计
  ├─ 检查紧急任务
  └─ 保存系统状态
```

## ⚙️ 高级配置

### 修改心跳间隔

编辑 `AI_Roland/system/daemon.py`：

```python
self.heartbeat_interval = 60  # 改为其他秒数
```

### 修改历史显示数量

编辑 `AI_Roland/system/session_start.py`：

```python
history = self.load_chat_history(limit=3)  # 改为其他数字
```

### 查看实时日志

```bash
# Windows PowerShell
Get-Content AI_Roland\logs\daemon_YYYYMMDD.log -Wait

# 或使用 tail 工具
tail -f AI_Roland/logs/daemon_YYYYMMDD.log
```
