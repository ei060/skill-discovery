# 浏览器控制器使用指南

## 功能概述

**版本**：v2.0
**日期**：2026-02-21
**作用**：提供持久化浏览器会话，支持远程操作和自动化

## 解决的问题

### 原有问题

在使用 `notebooklm login` 时：
- ❌ 浏览器打开后立即关闭
- ❌ 无法完成 Google 登录授权
- ❌ Session 无法保存，需要重复登录
- ❌ 没有可复用的浏览器自动化框架

### 解决方案

**浏览器控制器**：基于 Playwright 的持久化浏览器框架

- ✅ 浏览器稳定启动和关闭
- ✅ Session 自动保存（cookies + localStorage）
- ✅ 支持交互模式和程序控制
- ✅ 可复用于各种 Web 自动化任务

## 系统架构

```
浏览器控制器（BrowserController）
    ├── start() - 启动浏览器
    ├── goto(url) - 导航到 URL
    ├── click(selector) - 点击元素
    ├── fill(selector, value) - 填写表单
    ├── screenshot() - 截图
    ├── get_url() - 获取当前 URL
    ├── get_text() - 获取页面文本
    ├── evaluate(script) - 执行 JavaScript
    ├── run_interactive() - 交互模式
    └── stop() - 停止并保存 session
```

## 使用方式

### 方式1：交互模式（推荐新手）

**文件**：`浏览器控制器.bat`

**步骤**：
1. 双击 `浏览器控制器.bat`
2. 浏览器自动打开并显示 Google
3. 可以手动操作浏览器
4. 按 Ctrl+C 停止（session 自动保存）

**适用场景**：
- 需要手动完成登录
- 浏览器操作
- 快速测试

### 方式2：NotebookLM 登录助手

**文件**：`NotebookLM登录助手.bat`

**步骤**：
1. 双击 `NotebookLM登录助手.bat`
2. 浏览器打开 Google 登录页面
3. 手动完成 Google 账号登录
4. 登录成功后自动验证
5. Session 自动保存

**适用场景**：
- NotebookLM 首次授权
- 重新登录 Google
- 验证登录状态

### 方式3：程序控制（高级用户）

**代码示例**：
```python
from browser_controller import BrowserController

# 创建控制器
controller = BrowserController()

try:
    # 启动浏览器
    controller.start(headless=False, channel="chrome")

    # 导航到 URL
    controller.goto("https://www.google.com")

    # 填写搜索框
    controller.fill("textarea[name='q']", "AI Roland")

    # 截图
    controller.screenshot()

    # 获取页面文本
    text = controller.get_text()

finally:
    # 停止并保存 session
    controller.stop()
```

**适用场景**：
- 批量操作
- 自动化脚本
- 定时任务

## 功能详解

### 1. 启动浏览器

```python
controller = BrowserController()
controller.start(headless=False, channel="chrome")
```

**参数**：
- `headless`: False = 显示浏览器, True = 后台运行
- `channel`: "chrome" | "msedge" | "chromium"

**特性**：
- 自动加载保存的 session
- 自动保存 cookies 和 localStorage
- 窗口大小：1280x720

### 2. 导航操作

```python
# 导航到 URL
controller.goto("https://www.google.com")

# 等待导航完成
controller.wait_for_navigation()

# 等待元素出现
controller.wait_for_selector("#my-element")
```

### 3. 页面交互

```python
# 点击元素
controller.click("button.submit")

# 填写表单
controller.fill("input[name='email']", "user@example.com")

# 模拟打字（慢速）
controller.type_text("input[name='message']", "Hello", delay=100)
```

### 4. 获取信息

```python
# 获取当前 URL
url = controller.get_url()

# 获取页面文本
text = controller.get_text()

# 获取特定元素文本
element_text = controller.get_text("#my-element")

# 执行 JavaScript
result = controller.evaluate("document.title")
```

### 5. 截图

```python
# 自动命名截图
path = controller.screenshot()

# 指定路径
controller.screenshot("D:/screenshots/mypage.png")
```

### 6. 交互模式

```python
# 启动浏览器后进入交互模式
controller.run_interactive()
```

**效果**：
- 浏览器保持运行
- 可以手动操作
- 按 Ctrl+C 停止
- Session 自动保存

## Session 持久化

### 保存位置

```
AI_Roland/browser_data/
    └── storage_state.json  # cookies, localStorage, sessionStorage
```

### 保存时机

1. **自动保存**：调用 `stop()` 时自动保存
2. **内容**：
   - Cookies
   - localStorage
   - sessionStorage

### 加载时机

1. **自动加载**：调用 `start()` 时自动加载
2. **效果**：
   - 保持登录状态
   - 保留网站设置
   - 恢复购物车等

## NotebookLM 登录流程

### 完整流程

1. **启动登录助手**：
   ```
   双击：NotebookLM登录助手.bat
   ```

## 常见问题

### Q1: 浏览器打开后立即关闭

**原因**：使用了 `launch_persistent_context`

**解决**：已改用普通 `launch` + 手动保存 session

### Q2: Session 未保存

**检查**：
```bash
dir AI_Roland\browser_data\storage_state.json
```

**如果不存在**：
- 确保调用了 `stop()`
- 检查文件权限
- 查看错误信息

### Q3: 如何重新登录？

**方法1**：删除 session 文件
```bash
del AI_Roland\browser_data\storage_state.json
```

**方法2**：使用无 session 启动
```python
controller = BrowserController()
# 临时重命名 storage_state.json 后再启动
```

### Q4: 可以使用 Edge 吗？

**可以**：
```python
controller.start(headless=False, channel="msedge")
```

### Q5: 如何后台运行？

**设置 headless=True**：
```python
controller.start(headless=True)
```

### 示例2：批量截图

```python
from browser_controller import BrowserController

    "https://www.google.com",
    "https://www.github.com",
    "https://www.wikipedia.org",
]

controller = BrowserController()
controller.start(headless=True)  # 后台模式

for i, url in enumerate(urls):
    controller.goto(url)
    controller.screenshot(f"screenshot_{i}.png")

controller.stop()
```
