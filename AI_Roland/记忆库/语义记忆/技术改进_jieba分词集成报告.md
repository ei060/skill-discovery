## 改动对比

### v1.0（原版）

```python
def tokenize(self, text: str) -> List[str]:
    """简单的分词（中文和英文）"""
    text = re.sub(r'[^\w\s\u4e00-\u9fff]', ' ', text)
    tokens = []
    for word in text.split():
        # 如果是中文，每个字作为一个 token
        if any('\u4e00' <= char <= '\u9fff' for char in word):
            tokens.extend(list(word))  # ❌ 按字分词
        else:
            tokens.append(word.lower())
    return tokens
```

**问题**：
- ❌ 中文按字分词："记忆系统" → ["记", "忆", "系", "统"]
- ❌ 无法识别词语边界
- ❌ 搜索"记忆"会匹配到"记忆"、"纪念"、"记录"等所有包含"记"字的词

### v2.0（改进版）

```python
def tokenize(self, text: str) -> List[str]:
    """使用 jieba 分词（中文和英文）"""
    text = re.sub(r'[^\w\s\u4e00-\u9fff]', ' ', text)

    # 使用 jieba 分词
    tokens = jieba.lcut(text)

    # 过滤和清理
    filtered_tokens = []
    for token in tokens:
        token = token.strip()
        if len(token) <= 1:
            continue
        if token.isdigit():
            continue
        if not any(char.isalnum() or '\u4e00' <= char <= '\u9fff' for char in token):
            continue
        if not any('\u4e00' <= char <= '\u9fff' for char in token):
            token = token.lower()
        filtered_tokens.append(token)

    return filtered_tokens
```

**改进**：
- ✅ 中文按词分词："记忆系统" → ["记忆", "系统"]
- ✅ 准确识别词语边界
- ✅ 过滤单字符和纯数字
- ✅ 搜索"记忆"只匹配包含"记忆"这个词的内容

## 技术细节

### jieba 分词特点

1. **精准模式**：默认模式，精确切割
2. **全模式**：扫描所有可能的词语
3. **搜索引擎模式**：在精准模式基础上，对长词再次切分

**我们使用**：精准模式（`jieba.lcut()`）

### 过滤规则

```python
# 1. 过滤单字符
if len(token) <= 1:
    continue

# 2. 过滤纯数字
if token.isdigit():
    continue

# 3. 过滤纯标点
if not any(char.isalnum() or '\u4e00' <= char <= '\u9fff' for char in token):
    continue

# 4. 英文转小写
if not any('\u4e00' <= char <= '\u9fff' for char in token):
    token = token.lower()
```

1. **首次加载慢**：
