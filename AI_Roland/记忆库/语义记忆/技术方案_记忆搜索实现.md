# 记忆搜索实现方案

## 概念

基于语义相似度的智能记忆检索系统，支持对话历史、日记、语义记忆的多维度搜索。

## 当前实现

### 算法：TF-IDF + 余弦相似度

**工作流程：**
```
文档加载 → 分词 → TF-IDF计算 → 查询向量化 → 相似度计算 → 排序返回
```

### 核心组件

#### 1. 文档加载器
支持三种数据源：
- 对话历史（对话历史.md）
- 日记（日记/*.md）
- 语义记忆（记忆库/语义记忆/*.md）

#### 2. 分词器
- 支持中文（按字分词）
- 支持英文（按空格分词）
- 移除特殊字符

#### 3. TF-IDF 计算器
- TF（词频）：词在文档中出现的频率
- IDF（逆文档频率）：词的区分能力
- TF-IDF = TF × IDF

#### 4. 相似度计算
- 余弦相似度：向量夹角的余弦值
- 取值范围：[0, 1]
- 值越大越相似

### 性能优化

#### 文档缓存
```python
if self._documents_cache is not None:
    return self._documents_cache
```
避免重复加载文档

#### TF-IDF 缓存
```python
if self._tfidf_cache is not None:
    return self._tfidf_cache
```
避免重复计算 TF-IDF

## 升级路径

### Level 1: Sentence Transformers（推荐）

**优势：**
- 更好的语义理解
- 支持多语言
- 预训练模型

**实现：**
```python
from sentence_transformers import SentenceTransformer

model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2')

# 编码文档
embeddings = model.encode([doc["content"] for doc in documents])

# 编码查询
query_embedding = model.encode(query)

# 计算相似度
from sklearn.metrics.pairwise import cosine_similarity
similarities = cosine_similarity([query_embedding], embeddings)[0]
```

### Level 2: ChromaDB

**优势：**
- 本地向量数据库
- 自动持久化
- 支持过滤

**实现：**
```python
import chromadb

client = chromadb.PersistentClient(path="./memory_db")
collection = client.get_or_create_collection("memories")

# 添加文档
collection.add(
    documents=[doc["content"] for doc in documents],
    metadatas=[{"type": doc["type"]} for doc in documents],
    ids=[doc["path"] for doc in documents]
)

# 搜索
results = collection.query(
    query_texts=[query],
    n_results=5
)
```

### Level 3: FAISS

**优势：**
- 高性能相似度搜索
- 支持大规模数据
- GPU 加速

**适用场景：**
- 文档数量 > 10,000
- 需要毫秒级响应
- 有 GPU 资源

**当前测试（8个文档）：**
- 文档加载：< 1秒
- TF-IDF 计算：< 1秒
- 搜索查询：< 1秒

**预估（100个文档）：**
- 文档加载：~2秒
- TF-IDF 计算：~3秒
- 搜索查询：< 1秒（有缓存）
