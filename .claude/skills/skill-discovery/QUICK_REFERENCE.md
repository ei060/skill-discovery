# Skill Discovery - 快速参考卡

## 🎯 这是什么？

一个能**自动发现最新工具和技能**的 skill。AI 会自己判断何时使用，不需要你明确请求。

## ⚡ 快速开始

### 你什么都不用做！

当你说这些时，AI 会自动触发：
- "我需要自动化 [某事]"
- "用 [某工具] 做 [某事]"
- "我要做一个 [某项目]"
- 或者任何提到技术栈/工具的时候

### 手动触发（可选）

```
"搜索最新的 AI 工具"
"有什么新的浏览器自动化框架？"
"skill-discovery: 查找 [领域]"
```

## 🔍 支持的领域

| 领域 | 关键词示例 |
|------|-----------|
| 浏览器自动化 | puppeteer, playwright, selenium, 爬虫 |
| AI 智能体 | openclaw, claude, ai agent, chatbot |
| Python 脚本 | python, 脚本, automation |
| API 集成 | api, rest, graphql, webhook |
| DevOps 工具 | docker, kubernetes, ci/cd |
| 数据分析 | pandas, jupyter, visualization |

## 🚀 典型场景

### 场景 1：你不懂技术

```
你: "我需要自动登录网站"
AI: [自动搜索] → 发现 Playwright → 帮你实现
```

### 场景 2：你提了旧工具

```
你: "用 Selenium 做爬虫"
AI: [搜索发现] → Playwright 更好 → 推荐并解释
```

### 场景 3：规划新项目

```
你: "我要做个抢票工具"
AI: [调研] → 推荐技术栈 → 开始实现
```

## 🎨 工作原理

```
你说话
  ↓
AI 检测关键词
  ↓
检查缓存（是否过期？）
  ↓
搜索：GitHub + Reddit + Web
  ↓
合并 + 去重 + 评分
  ↓
更新缓存
  ↓
推荐最佳工具
```

```

```

## 📊 缓存策略

- **热门领域**（AI 工具）: 6 小时
- **中频领域**（浏览器自动化）: 12 小时
- **冷门领域**（脚本工具）: 24 小时

## 💡 核心特点

### 1. AI 主动判断
- AI 自己决定何时搜索
- 你不需要懂技术术语
- 发现更好的方案会自动推荐

### 2. 透明决策
- 告诉你用了什么工具
- 解释为什么用这个
- 你可以学习或覆盖

### 3. 持续学习
- 每次搜索更新知识库
- 缓存积累历史数据
- 越用越聪明

## 🛡️ 权限说明

**AI 有自主权**：
- ✅ 自动搜索相关工具
- ✅ 选择最佳方案
- ✅ 覆盖你的次优选择
- ⚠️ 但你可以强制覆盖 AI

**示例**：
```
你: "必须用 Selenium"
AI: "好的，但我建议 Playwright 因为..."
    [使用 Selenium]
```

## 🔧 技术细节

### 数据源
- **GitHub**: 开源项目（60次/小时，无认证）
- **Reddit**: 社区讨论（30次/分钟）
- **WebSearch**: 技术文章（无限制）

### Fallback 策略
- GitHub 限流 → WebSearch with "site:github.com"
- Reddit 失败 → 继续 GitHub + Web
- 全部失败 → 显示缓存 + 手动建议

## 🎓 最佳实践

### ✅ 推荐做法
- 直接说你的需求，不要指定工具
- 让 AI 推荐方案
- 信任 AI 的判断（它比你更了解工具生态）

### ❌ 避免做法
- 固定使用旧工具（"必须用 Selenium"）
- 忽略 AI 的推荐
- 自己搜索工具（让 AI 做）
